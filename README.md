# AI自动交易系统 - 说明文档

## 概述

本程序是受NOF1.AI启发，基于Web的AI自动交易演示系统，集成了币安交易所的实时数据、AI分析决策和模拟交易功能。系统采用三列式布局设计，提供全面的市场监控和交易执行能力。

实盘程序已推出，详见https://github.com/jimk2024/Aitrader100

## 主要功能

### 1. 实时市场数据监控
- **顶部价格栏**: 实时显示BTC/USDT、ETH/USDT、SOL/USDT、BNB/USDT、DOGE/USDT、XRP/USDT等主要加密货币价格
- **底部价格栏**: 重复显示关键交易对价格，便于随时查看
- **实时连接状态**: 显示与币安服务器的连接状态

### 2. 多交易对数据展示（第一列）
- 每个交易对独立显示面板
- 实时更新市场深度、交易量、价格变化等数据
- 支持BTCUSDT、ETHUSDT、SOLUSDT、BNBUSDT、DOGEUSDT、XRPUSDT

### 3. AI交互与分析模块（第二列）

#### AI配置
- **AI提供商选择**: 支持Gemini 2.5 Flash（免费）、OpenAI、Claude（待定）
- **API密钥管理**: 安全保存AI服务API密钥
- **策略规则输入**: 自定义交易策略规则

#### 数据分析
- **数据选择**: 可选择要分析的交易对数据
- **历史规则管理**: 保存和加载历史策略规则
- **AI优化提示词**: 自动优化交易策略提示词

#### 自动化功能
- **单次发送**: 手动触发AI分析
- **定时发送**: 支持3分钟、5分钟、10分钟、30分钟、60分钟自动分析间隔
- **自动执行**: 跟随AI分析结果自动执行交易

### 4. 合约模拟交易（第三列）

#### 账户管理
- **初始余额设置**: 可配置模拟账户初始资金
- **实时账户信息**: 显示余额、账户权益、保证金使用情况
- **盈亏统计**: 未实现盈亏和已实现盈亏跟踪

#### 交易功能
- **持仓管理**: 当前持仓情况展示
- **委托管理**: 当前委托订单管理
- **交易历史**: 历史交易记录
- **一键操作**: 撤销全部委托、平仓、重置模拟盘

#### 自动化交易
- **AI自动操盘**: 启用AI自动交易功能
- **收益曲线图**: 可视化账户余额变化

## 技术架构

### 前端技术
- **HTML5/CSS3**: 现代化响应式界面设计
- **JavaScript**: 原生JS实现核心逻辑
- **WebSocket**: 实时数据连接币安交易所
- **Canvas**: 账户收益曲线可视化

### 数据集成
- **币安API**: 实时市场数据获取
- **币安期货API**: 合约交易数据
- **多端点支持**: 自动故障转移和负载均衡

### AI集成
- **Gemini API**: 免费AI分析服务
- **多模型支持**: 可扩展支持其他AI提供商

## 安装与使用

### 运行要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 网络连接（访问币安API和AI服务）

### 快速开始
1. 直接打开 `demo.html` 文件
2. 系统自动连接币安交易所获取实时数据
3. 配置AI服务API密钥
4. 设置交易策略规则
5. 开始监控市场或启用模拟交易

### AI配置步骤
1. 获取Gemini或其他AI服务的API密钥
2. 在AI交互模块中输入API密钥并保存
3. 制定交易策略规则
4. 选择要分析的交易对数据
5. 启动AI分析功能

## 安全特性

- **本地存储**: API密钥和配置信息本地保存
- **加密传输**: 与交易所和AI服务的加密通信
- **模拟环境**: 交易操作在模拟环境中进行，无真实资金风险

## 界面特性

### 响应式设计
- **桌面端**: 三列布局，最大化信息展示
- **移动端**: 自适应单列布局，支持横向滚动
- **暗色主题**: 币安风格的暗色界面，减少眼睛疲劳

### 用户体验
- **实时更新**: 价格和数据自动刷新
- **状态指示**: 清晰的操作状态反馈
- **错误处理**: 完善的错误提示和恢复机制

## 注意事项

1. **模拟交易**: 当前为模拟环境，所有交易操作不会使用真实资金
2. **API限制**: 注意币安API的调用频率限制
3. **网络要求**: 需要稳定的网络连接获取实时数据
4. **浏览器兼容**: 建议使用最新版本浏览器
